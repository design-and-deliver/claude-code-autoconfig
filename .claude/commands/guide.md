# Autoconfig Guide

Open the interactive guide to learn about Claude Code Autoconfig.
```bash
# macOS
open .claude/guide/autoconfig.guide.html

# Linux
xdg-open .claude/guide/autoconfig.guide.html

# Windows
start .claude/guide/autoconfig.guide.html
```

The guide covers:
1. Project structure overview
2. CLAUDE.md — the entry point
3. settings.json — permissions & security
4. Slash commands — on-demand workflows
5. Rules — path-scoped context

**Tip:** You can also just open `.claude/guide/autoconfig.guide.html` directly in your browser.
